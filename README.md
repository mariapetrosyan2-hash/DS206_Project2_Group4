# DS206_Project2

## How to Run the Pipeline

1. Make sure the database and required tables already exist.
2. DDL scripts are not part of the pipeline and should be executed separately before running the ETL process.
3. Open the project folder in VS Code.
4. Configure the database connection settings if needed.
5. Run the pipeline scripts from the pipeline_dimensional_data folder.
6. Check the logging output to confirm whether the process completed successfully.

## Logging

- Logging is implemented without print statements.
- Logs are stored separately for monitoring and debugging purposes.
